Theme Switcher Test Page

Instructions:

This is a standalone test page to verify that the theme switcher displays correctly.

You should see the theme switcher in the top-right corner of this page.

If you can see the theme switcher on this page but not in your main application:

  • There might be CSS conflicts in your main application
  • Another element might be overlapping or hiding the theme switcher
  • There could be JavaScript errors preventing proper initialization

To troubleshoot further:

  1. Open the browser's developer tools (F12)
  2. Check the console for any JavaScript errors
  3. Inspect the theme-switcher element to see if it's being rendered correctly
  4. Look for CSS rules that might be affecting its visibility
